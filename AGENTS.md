# Project Instructions

Before working in this project, read `CLAUDE.md`.

Follow the conventions and warnings in `CLAUDE.md` unless the user explicitly says otherwise.

When adding or changing dependencies for AI lectures, update this project's
Conda environment definition (`ai/environment.yml`) and make sure the actual
`ai` Conda environment is updated too. Do not install packages only into a
parent or unrelated environment; `quarto preview` for this project must work
from the ai environment.
